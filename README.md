# UX OS

UX OS is a personal UX project workflow workspace built with Next.js. Deployed on Vercel with Supabase for data, auth, and file storage.

Before changing visual design, color, layout rhythm, component states, or interaction styling, read:

- `PRODUCT.md` for product intent and UX principles.
- `DESIGN.md` for the Konan-inspired paper-blue palette, token roles, component rules, and change workflow.

## Getting Started

First, run the development server:

```bash
npm.cmd run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Quality Checks

```bash
npm.cmd run lint
npm.cmd run build
```

## Deploy on Vercel

```bash
vercel.cmd deploy --prod --yes
```
