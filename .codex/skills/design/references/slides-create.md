# Slides Workflow

Use the local slide workflow bundled with `design-system` and `design` to create persuasive HTML slides using design tokens, Chart.js, and the slide knowledge base.

## Entry Points

- `references/slides-strategies.md`
- `references/slides-layout-patterns.md`
- `references/slides-copywriting-formulas.md`
- `references/slides-html-template.md`

## When To Use

- investor decks
- product narrative slides
- data-driven slides
- slide systems that need copy, layout, and chart guidance

## Workflow

1. Search strategies, layouts, and copy formulas.
2. Choose the slide pattern before writing content.
3. Generate or update the slide HTML.
4. Validate the output with the slide token validator if tokens are involved.
